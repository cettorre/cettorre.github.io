Dice Game Layered Pattern Jun 2018

GitHub project

Description:
Application simulating a dice game. When the user enters in the application, before starting the game, he has to introduce a player name. To win the game the sum of the dices has to be 7. After rolling the dices the application calculate the percentage of wins. For every game the results will be stored into the databse. To see all the list of results the user has to access to the statistics screen where a list of results for every player will be displayed.

Technical information:
The aplpication follows a design pattern diveded into layers. The application layer contains the logic of the program and allows communication between View and Domain. The domain layer contains classes representing the application objects such as Player, Game and Dice. The view layer contains all the activities of the application. In the persistence layer there are the operation with the database. To transfer data between layers a DTO has been used.

Tags : #Layered Pattern, #MVC, #DTO







Android SQLite Database May 2018

GitHub project

Description :
Application that stores animals information and photo in a SQLite database. The main activity shows all the animals that have been entered into the database. From the add animal button you can add a new animal in the database with the animal information: the name of the animal, the race, if the animal has a chip or not and a photo compressed in base 64 format. From the list in the main activity you can select an item to see the information of the animal, his photo and the inserting date. Animals that have chip are shown in blue colour in the main activity list .

Technical information :
The application is using SQLite database. Photo are stored as strings in base64 format.

Tags : #SQLite






AndroidVideoPlayer Abr 2018

GitHub project

Description :
This application is a video player that works together with th plugin PluginAndroidVideoPlayer The plugin is collecting information while the video player is playing videos. To check how they are working you need to install the video player and plugin in the device. When the video ends, the plugin will open automatically showing the statistics of the player on the screen.

Tags : #ExoPlayer



From the main activity of the plugin is possible to start the following tasks:
-show statistics of the last video
-send http request on a new thread
-send http request using a service
-clear all the data stored into the application
